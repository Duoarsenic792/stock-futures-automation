## 🛠️ Troubleshooting

### The app does not open
- Make sure you downloaded the Windows file.
- Check that the file finished downloading.
- Extract the zip file before opening the app.
- Try running the app as an administrator.

### The screen is blank
- Wait a few seconds for data to load.
- Check your internet connection.
- Refresh the app if there is a reload button.
- Close and reopen the program.

### Prices do not update
- Check the network connection.
- Confirm that the data feed is enabled.
- Look for a pause or stop button in the interface.
- Restart the app if the feed stalls.

### Windows blocks the file
- Right-click the file and open **Properties**.
- If you see an unblock option, enable it.
- Try again from the extracted folder.

### The app closes at startup
- Reboot your PC.
- Make sure other market tools are not using the same data source.
- Try a fresh download from the repository page.
